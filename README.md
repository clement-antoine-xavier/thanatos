# Thanatos

Thanatos personifies peaceful death, son of Nyx, and acts as Hades' messenger to carry away souls. Hades reigns over the Underworld, watching over the dead without being death itself.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
